# simple-social-server

Super simple social app for learning purposes

## Backend

### Navigation
```bash
cd ./backend
```
### Installation
```bash
npm install
```

### Usage
```bash
npm start
``` 
## Frontend
### Navigation
```bash
cd ./frontend
```
### Installation
```bash
yarn
```
### Usage
```bash
yarn dev
```
